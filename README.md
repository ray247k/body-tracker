# body-tracker

## Intro
把記錄在 google spreadsheet 的體重管理資料用比較視覺畫的方式呈現

體重太羞恥了，先以體脂肪變化為主

也許加上一些事件紀錄的 tag

使用 chart.js 來做圖表顯示的部分，安裝了 vue-chartjs 的 npm 套件
